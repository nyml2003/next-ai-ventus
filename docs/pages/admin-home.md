# 管理首页设计

## 页面目标

让作者快速了解博客整体状态，提供常用操作的快捷入口。

## 用户场景

| 场景 | 用户行为 |
|------|---------|
| 每日查看 | 看一眼文章数量、最近有谁访问（P2） |
| 快速写作 | 点击「新建文章」开始写作 |
| 内容管理 | 点击「文章管理」进入列表 |

## 模块清单

### 1. Sidebar（侧边导航）

**BFF 模块：`adminSidebar`**（首屏模块）

**功能：**
- 博主头像和名称
- 主导航菜单
- 退出登录

**导航菜单：**
```
┌─────────────────┐
│   [头像]        │
│   张三          │
├─────────────────┤
│  📊 仪表盘  ←   │
│  📝 文章管理    │
│  🖼️ 图片管理    │
│  ⚙️ 设置        │
├─────────────────┤
│  🚪 退出登录    │
└─────────────────┘
```

**交互：**
- 当前页面高亮显示
- 点击菜单切换页面

### 2. Stats（数据统计）

**BFF 模块：`adminStats`**（首屏模块）

**功能：**
- 展示关键指标

**指标卡片：**
```
┌─────────────────────────────────────────┐
│  📄 文章总数      👁️ 总阅读量           │
│     15 篇            3,240 次           │
│                                         │
│  🏷️ 标签数量      📅 最近更新           │
│     8 个            2 天前              │
└─────────────────────────────────────────┘
```

**说明：**
- v0.1 阶段数据简单，仅展示数量
- P2 阶段增加图表（阅读量趋势、访问来源等）

### 3. QuickActions（快捷操作）

**说明**：纯静态组件，无 BFF 数据

**功能：**
- 最常用的操作一键直达

**按钮组：**
```
┌─────────────────────────────────────────┐
│  [+ 新建文章]  [📁 文章管理]  [⚙️ 设置]   │
└─────────────────────────────────────────┘
```

### 4. RecentPosts（最近文章）

**BFF 模块：`recentPosts`**（首屏模块）

**功能：**
- 展示最近编辑的 5 篇文章
- 快速查看和编辑

**列表结构：**
```
┌─────────────────────────────────────────┐
│  最近编辑                                │
├─────────────────────────────────────────┤
│  文章标题                 [编辑] [查看]  │
│  2 小时前 · 已发布                       │
├─────────────────────────────────────────┤
│  另一篇文章               [编辑] [查看]  │
│  1 天前 · 草稿                           │
├─────────────────────────────────────────┤
│  ...                                     │
├─────────────────────────────────────────┤
│  查看全部 →                              │
└─────────────────────────────────────────┘
```

**交互：**
- 点击文章标题进入编辑页
- 「查看」按钮在新窗口打开文章详情
- 底部「查看全部」跳转到文章管理页
