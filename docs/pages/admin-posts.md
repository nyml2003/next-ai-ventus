# 文章管理页设计

## 页面目标

集中管理所有文章，支持筛选、搜索、批量操作。

## 用户场景

| 场景 | 用户行为 |
|------|---------|
| 查找文章 | 通过搜索或筛选找到目标文章 |
| 批量管理 | 批量删除草稿、批量发布 |
| 修改文章 | 点击编辑进入编辑器 |
| 查看效果 | 点击预览在新窗口打开文章 |

## 模块清单

### 1. Sidebar（侧边导航）

与「管理首页」相同，当前选中「文章管理」

### 2. PageHeader（页面标题栏）

**功能：**
- 页面标题：文章管理
- 操作按钮：+ 新建文章

**布局：**
```
┌─────────────────────────────────────────┐
│  文章管理                     [+ 新建]  │
└─────────────────────────────────────────┘
```

### 3. FilterBar（筛选栏）

**功能：**
- 状态筛选：全部 / 已发布 / 草稿
- 标签筛选：下拉选择标签
- 搜索框：按标题搜索

**布局：**
```
┌─────────────────────────────────────────┐
│  [全部] [已发布] [草稿]     [标签 ▼]  [🔍 搜索]  │
└─────────────────────────────────────────┘
```

**交互：**
- 点击状态标签切换筛选
- 选择标签后仅展示该标签的文章
- 搜索实时过滤（或回车搜索）

### 4. PostTable（文章表格）

**功能：**
- 表格展示文章列表
- 支持选择行进行批量操作

**表头：**
| 选择框 | 标题 | 标签 | 状态 | 更新时间 | 操作 |
|--------|------|------|------|----------|------|

**行内容示例：**
```
┌─────────────────────────────────────────────────────────┐
│ ☐ │ 如何设计高性能博客   │ [架构][Go] │ 🟢已发布 │ 2小时前 │ [编辑][删除] │
│ ☐ │ 我的2024技术总结     │ [生活]     │ 🟡草稿   │ 1天前   │ [编辑][删除] │
└─────────────────────────────────────────────────────────┘
```

**状态标识：**
- 🟢 已发布：对外可见
- 🟡 草稿：仅管理员可见
- ⚫ 归档：已下线（P2）

**交互：**
- 点击标题预览文章
- 点击「编辑」进入编辑器
- 点击「删除」二次确认后删除

### 5. BatchActions（批量操作）

**功能：**
- 选中多行后显示批量操作栏

**布局：**
```
┌─────────────────────────────────────────┐
│  已选择 3 项    [批量发布] [批量删除] [取消] │
└─────────────────────────────────────────┘
```

### 6. Pagination（分页器）

**功能：**
- 文章较多时分页展示
- 每页默认 20 条

**布局：**
```
┌─────────────────────────────────────────┐
│  共 45 篇    [上一页] 1 2 3 ... 5 [下一页] │
└─────────────────────────────────────────┘
```

## 交互细节

### 删除确认

- 点击「删除」弹出确认对话框
- 确认后才执行删除操作
- 删除后 Toast 提示「已删除」

### 批量操作

1. 用户勾选多行
2. 底部弹出 BatchActions 栏
3. 点击批量操作按钮
4. 二次确认后执行
5. 操作完成后刷新列表

### 搜索体验

- 输入时实时过滤（防抖 300ms）
- 高亮匹配的关键词
- 搜索结果按更新时间倒序
